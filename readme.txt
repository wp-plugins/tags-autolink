== Description ==

Tags Autolink is a SEO plugin which automatically turns tag names, when used in posts and pages, in links to the corresponding tag URLs. Keeps the original words case in text. It's particulary useful for SEO, for it creates an huge amount of internal links.

== Installation ==

1. Upload the tags-autolink folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're done.

== Frequently Asked Questions ==

= How many instances of the page names are turned into links? =

Each and all instance.
